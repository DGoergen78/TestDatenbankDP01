PostgreSQL-Datenbank in GitHub Codespaces starten

In dieser Anleitung startest du eine fertige PostgreSQL-Testdatenbank mit Docker. Die Datenbank enthaelt Tabellen fuer Mitglieder, Buchungen, Kurse und Trainer.

Repository: https://github.com/DGoergen78/TestDatenbankDP01

Wichtig: Du musst das Repository nicht vorher clonen. Wenn du Codespaces verwendest, lädt GitHub den Code automatisch.

Voraussetzung

  • Du brauchst einen GitHub-Account.
  • Du musst bei GitHub angemeldet sein.

Codespace starten

  1. Oeffne das Repository auf GitHub.
  2. Klicke auf den gruenen Button Code.
  3. Waehle den Reiter Codespaces.
  4. Klicke auf Create codespace on main.
  5. Warte, bis Codespaces vollstaendig geladen ist.

Repository oeffnen

Datenbank starten

Oeffne in Codespaces das Terminal und fuehre diesen Befehl aus:

docker compose up -d

Pruefe danach, ob die Container laufen:

docker compose ps

Datenbank im Browser oeffnen

  1. Oeffne unten in Codespaces den Bereich Ports.
  2. Suche den Port 8080.
  3. Klicke beim Port 8080 auf das Browser- oder Globus-Symbol.
Nicht Port 5432 im Browser oeffnen. Port 5432 ist der PostgreSQL-Datenbankport. Die Browseroberflaeche laeuft ueber Port 8080.

Login in Adminer

In der Browseroberflaeche meldest du dich mit diesen Daten an:

System PostgreSQL
Server postgres
Username kursuser
Password kurspass
Database kursdatenbank

Erste SQL-Befehle

Nach dem Login kannst du SQL-Befehle testen, zum Beispiel:

SELECT * FROM mitglied;
SELECT COUNT(*) FROM mitglied;
SELECT COUNT(*) FROM kurs;
SELECT COUNT(*) FROM trainer;
SELECT COUNT(*) FROM buchung;

Weitere SQL-Beispiele

Kurse von Trainer Markus Weber mit Anzahl der Buchungen:

SELECT K.K_Name, T.T_Nachname, T.T_Vorname, COUNT(B.K_ID) AS Anzahl_Buchungen
FROM Kurs K
INNER JOIN Trainer T ON K.T_ID = T.T_ID
LEFT JOIN Buchung B ON K.K_ID = B.K_ID
WHERE T.T_Vorname = 'Markus' AND T.T_Nachname = 'Weber'
GROUP BY K.K_Name, T.T_Nachname, T.T_Vorname
ORDER BY Anzahl_Buchungen DESC;

Alle Kurse mit Trainername, Startzeit und Preis:

SELECT
  K.K_Name,
  K.K_Startzeit,
  K.K_Endzeit,
  K.K_Preis,
  T.T_Vorname,
  T.T_Nachname
FROM Kurs K
INNER JOIN Trainer T ON K.T_ID = T.T_ID
ORDER BY K.K_Startzeit;

Alle Buchungen mit Mitglied, Kurs und Buchungsstatus:

SELECT
  M.M_Vorname,
  M.M_Nachname,
  K.K_Name,
  B.B_Datum,
  B.B_Status
FROM Buchung B
INNER JOIN Mitglied M ON B.M_ID = M.M_ID
INNER JOIN Kurs K ON B.K_ID = K.K_ID
ORDER BY B.B_Datum DESC;

Anzahl der Mitglieder pro Tarif:

SELECT
  M_Tarif,
  COUNT(*) AS Anzahl_Mitglieder
FROM Mitglied
GROUP BY M_Tarif
ORDER BY Anzahl_Mitglieder DESC;

Durchschnittlicher Kurspreis pro Trainer:

SELECT
  T.T_Vorname,
  T.T_Nachname,
  ROUND(AVG(K.K_Preis), 2) AS Durchschnittspreis
FROM Trainer T
INNER JOIN Kurs K ON T.T_ID = K.T_ID
GROUP BY T.T_Vorname, T.T_Nachname
ORDER BY Durchschnittspreis DESC;

Die 10 meistgebuchten Kurse:

SELECT
  K.K_Name,
  COUNT(B.B_ID) AS Anzahl_Buchungen
FROM Kurs K
LEFT JOIN Buchung B ON K.K_ID = B.K_ID
GROUP BY K.K_Name
ORDER BY Anzahl_Buchungen DESC
LIMIT 10;

Mitglieder, die den Tarif Premium haben:

SELECT
  M_ID,
  M_Vorname,
  M_Nachname,
  M_Email,
  M_Tarif
FROM Mitglied
WHERE M_Tarif = 'Premium'
ORDER BY M_Nachname, M_Vorname;

Buchungen mit dem Status bezahlt:

SELECT
  B.B_ID,
  M.M_Vorname,
  M.M_Nachname,
  K.K_Name,
  B.B_Datum,
  B.B_Status
FROM Buchung B
INNER JOIN Mitglied M ON B.M_ID = M.M_ID
INNER JOIN Kurs K ON B.K_ID = K.K_ID
WHERE B.B_Status = 'bezahlt'
ORDER BY B.B_Datum;

Kurse mit freien Plaetzen:

SELECT
  K.K_Name,
  K.K_MaxTeilnehmer,
  COUNT(B.B_ID) AS Anzahl_Buchungen,
  K.K_MaxTeilnehmer - COUNT(B.B_ID) AS Freie_Plaetze
FROM Kurs K
LEFT JOIN Buchung B ON K.K_ID = B.K_ID
GROUP BY K.K_Name, K.K_MaxTeilnehmer
ORDER BY Freie_Plaetze DESC;

Wenn etwas nicht funktioniert

Starte die Container neu:

docker compose down
docker compose up -d

Wenn weiterhin ein Fehler auftritt, pruefe die laufenden Container:

docker compose ps

Merksatz

  • Port 8080 ist fuer den Browser.
  • Port 5432 ist fuer PostgreSQL und wird nicht direkt im Browser geoeffnet.
